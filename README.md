# 宠物乐园 - 教育游戏化养成小程序

## 功能介绍

这是一个教育游戏化微信小程序，核心功能：
- 🐾 **宠物领养养成** - 领养属于你的宠物，三种宠物类型可选
- ✅ **任务系统** - 完成任务获得经验和属性奖励，帮助宠物升级
- ➕ **自定义任务** - 通过两位数乘法验证后可以创建自定义任务
- 🏫 **班级功能** - 创建班级，通过6位加入码加入，查看班级排行榜
- 📊 **属性系统** - 饱食度、心情值、清洁度，会随时间自然衰减

## 部署步骤

### 1. 在微信开发者工具中打开项目
1. 打开微信开发者工具
2. 导入项目，选择当前目录
3. 填入你的小程序 AppID（在 `project.config.json` 中修改）

### 2. 开通云开发
1. 在开发者工具中点击「云开发」
2. 创建云开发环境，记住环境 ID
3. 在 `app.js` 中填入你的环境 ID：
```javascript
wx.cloud.init({
  env: '你的环境ID',
  traceUser: true,
})
```

### 3. 创建数据库集合
在云开发控制台 → 数据库 → 集合管理，创建以下 5 个集合：
- `users` - 用户信息
- `pets` - 宠物信息
- `tasks` - 任务列表
- `userTasks` - 用户任务完成记录
- `classes` - 班级信息

### 4. 部署云函数
右键点击 `cloudfunctions` 下的每个文件夹（`login`、`checkAnswer`、`addTask`），选择「上传并部署：云端安装依赖」，分别部署三个云函数。

### 5. 初始化默认任务
可以在 `tasks` 集合中添加一些默认任务，例如：

| 标题 | 描述 | 经验奖励 | 饥饿奖励 | 心情奖励 | classId |
|------|------|---------|---------|---------|---------|
| 阅读30分钟 | 安静阅读半小时 | 15 | -15 | +10 | null |
| 做20道数学题 | 完成20道数学练习 | 20 | -10 | +5 | null |
| 背10个新单词 | 背诵记住10个新单词 | 15 | -10 | +8 | null |
| 帮做家务 | 帮助家长做家务 | 10 | -5 | +15 | null |
| 练习书法 | 练习15分钟书法 | 12 | -8 | +12 | null |

## 验证机制说明

自定义任务需要验证，验证方式是**答对一道两位数乘法题**：
- 打开创建任务页面会随机生成 `10-99 × 10-99` 的题目
- 答案在云函数验证，防止客户端作弊
- 答错会自动生成新题目，可以重试
- 答对后解锁任务创建表单

这满足了需求："简单验证，不需要角色区分"，只要会算两位数乘法就能创建自定义任务。

## 使用流程

1. 第一次打开小程序自动登录
2. 创建班级或输入加入码加入班级
3. 领养宠物，选择类型并命名
4. 在任务页面完成任务，获得奖励，升级宠物
5. 需要自定义任务时，去任务页面点击「+ 自定义」，答对乘法题后创建

## 数据库权限设置

建议在数据库权限设置中选择「仅创建者可读写」，这样用户只能修改自己的数据，保证数据安全。
